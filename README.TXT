PROJECT TITLE: Queue Lab
PURPOSE OF PROJECT: To learn about multi-level feedback queues and generally how to use queues.
VERSION or DATE: 7/20/2011
HOW TO START THIS PROJECT:  Run the Driver class' main method.
AUTHORS: Tobin deKorne' and Richard Stegman
USER INSTRUCTIONS: Simply run the Driver class!
DESIGN AND CLASSES USED: This program is run using the Driver class. The Driver class uses the MFQ class.
                            The MFQ class uses the ObjectQueue, CPU, Job, and LineWriter classes.
                            Finally, the CPU class uses the Job class.
EXTRA CREDIT:   This program outputs all of the following:
                Total number of jobs
                Total time of all jobs in the system
                Average response time
                Average turnaround time for the jobs
                Average waiting time
                Average throughput for the system as a whole
                Total CPU idle time
